
## Real-Estate 3D Marketplace

Demo App for Real Estate Marketplace with ability to add your custom 3D Model of the property, Empowered with custom AI-assistant and ability to promote your image on the top of the page for a promotion payment.
## Tech Stack

Next.js, Supabase, Three.js, TypeScript, React, Square API, OpenAI API


## Features

- CRUD of an advertisement of real estate
- Message the owner
- Chat With AI-assistant in case of aid-need
- Creating your own real-estate model
- Promote your real-estate advertisement
- Authentication
